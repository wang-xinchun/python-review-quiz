# 测试记录

## 1. 当前测试目标

第一轮测试重点验证核心业务逻辑和关键页面交互，而不是追求复杂界面样式：

1. 题目答案判断是否正确。
2. 题库能否初始化、新增和筛选。
3. 自测判分是否准确。
4. 练习记录和错题记录能否保存。
5. 自测页面是否避免默认选中答案，未完成答题时是否给出明确提示。
